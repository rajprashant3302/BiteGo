# Zomato Clone (Microservices + Kafka)

This is a Docker-based microservices project with:
- Multiple frontends (user, admin, vendor, driver)
- Backend services (auth, order, chat, earning)
- GraphQL Gateway
- Kafka + Zookeeper
- Redis + MongoDB
- NGINX reverse proxy

## Prerequisites
Make sure you have these installed:

- Docker Desktop (Windows / Mac / Linux)
- Git

## How to Run (One Command)

git clone https://github.com/rajprashant3302/BiteGo.git
cd BiteGo
docker-compose up -d --build

## to close server

docker-compose down
